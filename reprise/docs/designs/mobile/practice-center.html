Design Decisions
Mobile Adaptation Notes
Navigation
Back button → song detail. Ellipsis (···) → settings menu (speed, loop, pitch analysis). Section indicator in header when in section mode.
Lyrics Display
Prev/current/next shown vertically. Furigana above main text. Translations below. Tap = preview, double-tap = edit. Swipe L/R = prev/next line.
Touch Targets
All transport buttons ≥ 44×44pt. Record button is 60×60pt (primary action). Feedback buttons span full width for easy reach.
Recording State
Dynamic Island glows red. Red level meter in top bar. Active lyric turns red. Prev/next lines fade out. Progress bar shows playback position. Nav buttons disabled.
Countdown Overlay
Dim overlay covers full screen. Beat number large-centered. Beat dots show progression. Cancel button available. Record button turns amber with beat count.
Feedback Panel
Slides up from bottom after recording stops. "How'd that go?" with 3 actions: Listen (plays back with gain), Retry (delete & re-record), Next (accept & advance).
Section Mode
Section badge in header. All section lines displayed vertically with active line large. Section record button in transport. Translation shown for each line.
Edit Mode
Full-screen modal. Text input with keyboard. Live preview with tap-to-highlight. Special symbol row (♪ ↗ ↘ 息). Highlight palette at bottom. Done saves + dismisses.
Dark Mode
Background #0C0C10 (near-black). Borders #1A1A24. Text #F9FAFB → #6B7280 muted. Blue accents #3B82F6. Recording red carries through with reduced opacity glow.
Gestures (React Native)
Swipe left/right on lyrics area → prev/next line. Long-press lyric → enter edit mode. Swipe down on edit modal → discard. Pull down on transport → show speed sheet.
